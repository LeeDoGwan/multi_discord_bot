# discordBOT

## Project Definition

`discordBOT` is a Discord bot project built to learn Kotlin and design a practical server architecture.

The project is designed to manage multiple Discord bots through a single management system. Each bot focuses on its own business logic, while `motherBot` manages Discord connections, bot lifecycles, and communication between bots.

## Project Purpose

- Improve Kotlin application development skills
- Learn Discord bot development and Discord API integration
- Design a scalable bot management architecture
- Gain experience with real-time service operation and zero-downtime deployment
- Build a reusable Discord bot framework

## Project Goals

- Centrally manage Discord connections and bot lifecycles through `motherBot`
- Add and remove bots in real time without service interruption
- Allow each bot to maintain independent business logic
- Support message and event delivery between bots
- Provide bot status monitoring and control through a web management interface
- Provide a reusable framework for common bot functionality
